# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-02

### Added
- Initial release
- Config flow UI for county selection (42 counties + București)
- Fuel price sensors per county:
  - Benzină Standard 95
  - Motorină Standard
  - GPL Auto
  - Benzină Premium 98/100
  - Motorină Premium
- Cheapest station sensor with top 10 ranking
- Network comparison sensor (Petrom, OMV, Rompetrol, MOL, Lukoil, etc.)
- Data source: PretCarburant.ro (ANPC Monitorul Prețurilor)
- Update interval: configurable (default 2 hours)
- Full HACS support
- Romanian translations
